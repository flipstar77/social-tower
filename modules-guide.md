# The Tower - Complete Modules Guide

## Overview

Modules are powerful equipment pieces unlocked at **Tier 2, Wave 90**. There are 4 types of modules, each focusing on different aspects of gameplay. Modules can be leveled up, merged to higher rarities, and customized with sub-module effects to create powerful builds.

## Module Stats

Modules have **three types of stats:**

### 1. Main Effect
- **Only stat that increases with module level**
- Determines module's primary bonus (damage, health, coins, or UW damage)
- Scales based on module type and rarity

### 2. Sub-Module Effects
- **Up to 7 slots** (unlocked at levels 1, 41, 101, 141, 161, 201, 241)
- Independent rarity from the module itself
- Can be rerolled using reroll shards
- Effects can be locked to prevent rerolls (increases cost)

**Sub-Module Rarity Drop Rates:**
| Rarity | Chance |
|--------|--------|
| Common | 46.2% |
| Rare | 40% |
| Epic | 10% |
| Legendary | 2.5% |
| Mythic | 1.0% |
| Ancestral | 0.3% |

### 3. Unique Effects
- **Only available on naturally-pulled Epic modules**
- Rare modules merged to epic do NOT gain unique effects
- Provides special abilities beyond standard substats

## Merging Modules

### Merging Rules

**Important Notes:**
1. **Rare modules NEVER gain unique effects when merged to epic or higher**
2. **Rare modules keep their original name** (won't match naturally-pulled epics)
3. **Merging does NOT reset the current level** of the module
4. **Merging increases:**
   - Main stat value
   - Main stat scaling per level
   - Maximum level cap

### Merge Paths

**Common Modules:**
- Cannot be merged (remain common)
- Can only reach level 30
- Best used for shattering into shards

**Rare Modules:**
- Can merge to Legendary+ (max level 120)
- Will NOT gain unique effects
- Useful as wildcard merge materials

**Epic Modules (Naturally Pulled):**
- Can merge to Ancestral 5â˜… (max level 300)
- HAVE unique effects
- Most valuable long-term investment

### Ancestral Stars

Ancestral modules can be upgraded to **5 stars**:
- Each star requires a natural Epic+ module
- Each star increases the main effect
- Stars increase maximum level:
  - Ancestral (no stars): Level 200
  - Ancestral 1â˜…: Level 220
  - Ancestral 2â˜…: Level 240
  - Ancestral 3â˜…: Level 260
  - Ancestral 4â˜…: Level 280
  - Ancestral 5â˜…: Level 300

## Assist Modules

**Unlocked at:** Tier 19, Wave 40

Assist modules allow equipping a **second module to the same slot**, providing:
- Weaker versions of unique effect
- Reduced main effect
- Reduced substats

### Assist Module Costs

**Initial Unlock:** 1,000 stones per slot

**Unique Effect Rarity Upgrades:**
| Rarity | Stone Cost |
|--------|------------|
| Epic (unlock) | 1,000 |
| Legendary | 1,000 |
| Mythic | 1,200 |
| Ancestral | 1,400 |
| **Total** | **4,600 stones** |

**Main Effect / Substat Scaling:**
- Base: 1% effectiveness
- Can upgrade to 70% effectiveness
- **Total stones needed:** 8,073 stones

**Lab Upgrades:**
- 30 levels available
- Each level adds 1% effectiveness
- Max: +30% effectiveness

### Assist Module Restrictions
- Cannot equip modules with the **same name**
- Assist level is **separate** from main module level
- Does NOT bypass hard caps (98% defense, 90% CF slow, etc.)

## Module Strategy Tips

### Early Game (Tier 1-9)
1. Focus on **Common rarities** for basic stats
2. Save gems for more important upgrades (lab slots, cards)
3. Level modules to **level 41** for second substat slot

### Mid Game (Tier 10-15)
1. Start collecting **Epic modules** for unique effects
2. Focus on substats that synergize with your build
3. Begin merging Rare modules for better stats
4. Level key modules to **level 101** (3rd substat slot)

### Late Game (Tier 16-19)
1. Aim for **one Epic of each type**
2. Push Epic modules toward **Legendary/Mythic** rarities
3. Use Auto Roll to hunt for Mythic/Ancestral substats
4. Level main modules to **level 161** (5th substat slot)

### End Game (Tier 20+)
1. Work toward **Ancestral 5â˜…** modules
2. Optimize substats with Effect Bans
3. Unlock and upgrade **Assist Modules**
4. Level modules to **level 300** (maximum power)

## Key Vault Research (Module-Related)

From labs, the following can reduce module costs:
- **Module Shards Cost:** Up to -30% (30 levels)
- **Module Coin Cost:** Up to -30% (30 levels)
- **Common Drop Chance:** Up to +10% (10 levels)
- **Rare Drop Chance:** Up to +10% (10 levels)
- **Reroll Shards:** +100 per daily mission (50 levels)

**Unmerge Module:** Allows breaking modules back apart (1 level unlock)

**Shatter Shards:** Gain +5 shards per shatter (5 levels)
